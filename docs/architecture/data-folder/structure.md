# Structure

## Basic

```bash
data/
├── bin/
├── levelpacks/
├── mods/
├── runtime/
├── settings/
└── texturepacks/
```

## Full

```bash
data/
├── bin/
│   └── WCCACHE
├── levelpacks/
├── mods/
├── runtime/
│   └── discord/
├── settings/
│   ├── devoptions.json
│   ├── keybinds.json
│   └── settings.json
└── texturepacks/
```
